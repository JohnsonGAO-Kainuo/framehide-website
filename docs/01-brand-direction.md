# 01 · 品牌与视觉基调（FrameHide）

> 这份文档定下网站的“调性”，是后面所有页面设计的依据。先确认这一层，再谈页面。

---

## 1. 品牌定位

- **品牌名（暂定）**：**FrameHide** —— *frame*（取景框／画面）+ *hide*（皮革，亦含“收纳/守护”之意）。
- **一句话定位**：为热爱胶片质感与复古相机的人，做**有手感、有故事**的相机周边。
- **生意形态**：一人 / 小团队跨境（外贸），主市场为英语国家，**英文优先**。
- **首发品类**：富士 X100VI 真皮（PU）半套 / 底座保护套。
- **规划品类**（同一套视觉可复用）：
  - 皮革背带 / 腕带 / 绳套（自有调性，差异化强，利润好）
  - 相机兔笼（cage）——避开大牌正面竞争，选**白牌**高性价比款贴牌
  - 镜头盖、热靴盖、收纳袋等小配件

> 战略要点：**先用“皮套”建立品牌审美与信任**，再用同一视觉体系横向扩到背带/兔笼/小件。
> 网站架构从第一天就要“为多产品”预留，而不是写死单品。

---

## 2. 这是什么类型的网站？（已确认）

你的直觉是对的：**不做“带支付后端的完整独立站”**（购物车 / 库存 / 结账 / 退换全流程）。
刚起步、一人外贸，那套系统的运营负担远大于收益，也不利于建立品牌差异。

**确认形态：品牌 + 产品图鉴站，成交走线下询盘闭环**：

1. 网站把内容、故事、产品拍得**足够好看**，建立品牌与信任；
2. 客户有需求 → 通过 **WhatsApp / 邮件 / 社交媒体**直接联系我们；
3. 我们**人工报价 → 签合同** → 用 **Stripe 支付链接**（点链接即可付款）或 **银行转账** 收款；
4. 成交闭环发生在**网站之外**，网站上只需“咨询 / 获取报价”入口，**不放购物车/立即购买**。

> 这种“人对人 + Stripe Link”的方式最适合刚起步、订单量不大、且常涉及定制/批发报价的外贸。
> 等品类和流量起来，再平滑升级为完整电商。详见 `02-website-prd.md`。

## 3. 视觉基调：克制的高级感 × 复古胶片的人味

两组素材给出了清晰但**看似矛盾**的方向，我们把它融合成品牌的差异点：

| 来源 | 给出的信号 |
|---|---|
| `image/` 产品图 | 米色/奶油背景、原木、旧书、暖光、皮革质感 → **干净、克制、贵感、手作** |
| `style referece/` 风格图 | 复古胶片颗粒、褪色暖调、粗壮复古字体、meme 猫的幽默感 → **怀旧、胶片文化、有人味、Gen-Z** |

**融合策略 =「编辑感的高级底子」+「胶片质感的复古点缀」+「一点点幽默」**：

- **底子**：大量留白、克制的排版、温暖中性色，承接产品图的贵感（用 `high-end-visual-design` 技能）。
- **点缀**：轻微胶片颗粒/噪点叠层、复古标题字、做旧分割线，承接风格图的怀旧（用 `image-to-code` 技能落地）。
- **人味**：在“关于/社区”这类区块，用幽默语气和胶片文化梗，体现“一个人认真做东西”的真实感。
  - ⚠️ `style referece/` 里的 meme 猫**只作为情绪板（mood board）参考**，不直接上线（版权未知）。

## 4. 配色（草案，开发时落成设计变量）

以产品照背景与皮革色为锚点：

| 角色 | 颜色 | 用途 |
|---|---|---|
| 背景主色 | 奶油米色 `#F3E9DC` 附近 | 页面大底色，呼应产品图背景 |
| 纸张白 | `#FBF7F0` | 卡片/区块底 |
| 咖啡棕（品牌色） | `#6B4A2E` ~ `#3E2A1B` | 主按钮、标题强调、对应 Coffee 皮套 |
| 橄榄绿 | `#5C5A36` ~ `#6E7253` | 第二强调色，对应 Green 皮套 |
| 墨黑 | `#1E1B17` | 文字、对应 Black 皮套 |
| 复古红（点睛） | `#C8402F` 附近 | 极少量点缀，呼应相机快门红点 |

> 三个强调色刚好对应三款 SKU（Coffee / Green / Black），天然形成品牌色板。

---

## 5. 字体方向（草案）

- **标题**：带复古/编辑感的衬线或高对比无衬线（如 *Fraunces*、*Playfair Display*，或粗壮 grotesk 做复古海报感）。
- **正文**：干净易读的无衬线（如 *Inter* 的替代——避免太“AI 默认”，可用 *Geist* / *Schibsted Grotesk* / *Hanken Grotesk*）。
- **点缀**：风格图里那种**粗壮复古标题**用于个别大标语，制造“胶片海报”记忆点。

> 具体字体在开发阶段定稿；taste-skill 会刻意避开 “Inter + slate-900” 这类 AI 套路组合。

---

## 6. 语气 / 文案风格

- 英文为主，**简洁、自信、有手作温度**，少营销腔。
- 允许一点幽默与胶片文化梗（“shot on film, carried in leather”这类）。
- 关键信息（防护、精准开孔、电池快拆、三脚架接口）**讲清楚、讲可信**——外贸客户最看重“真不真、合不合身”。
