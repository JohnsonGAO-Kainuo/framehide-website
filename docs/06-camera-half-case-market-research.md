# 06 · 相机半套品类调研与验证框架

> 目标：判断 `X100VI / X100V / Ricoh GR / 复古相机周边` 是否值得作为一人外贸冷启动方向，并明确接下来 14 天怎么验证。

---

## 1. 当前结论

相机半套可以作为冷启动方向，但不能只押一个 `X100VI PU half case`。

更稳的定位是：

```text
Small wholesale accessories for compact, film, and retro-style camera shops.
```

首发产品仍然是：

```text
PU leather half case compatible with Fujifilm X100VI
```

但市场验证时要同时收集：

- 店铺是否对 `X100VI` 有兴趣；
- 店铺是否更关心 `X100V / Ricoh GR / film camera / straps / wrist straps`；
- 店铺是否愿意看样品、问价格、问 MOQ；
- 哪些店会回复，哪些店完全无反应。

---

## 2. 为什么这个品类有机会

### 2.1 热门相机带来周边需求

`Fujifilm X100VI`、`X100V`、`Ricoh GR` 这类相机有几个共同点：

- 用户愿意为外观、手感、复古感付费；
- 机身价格高，用户会买保护配件；
- 皮套、腕带、背带、软快门、镜头盖这类小件容易作为门店 add-on；
- 店铺可以把它们作为“顺手加购”的小产品，而不是高决策成本大件。

### 2.2 红海主机，周边仍有切入空间

相机本体是红海，授权、库存、品牌和资金门槛都高。

周边小件的优势：

- 起订低；
- 资金压力小；
- 可用样品试单；
- 店铺采购决策链较短；
- 不需要卖给终端消费者，可以先做 B2B 验证。

劣势：

- 容易同质化；
- 低价竞争明显；
- 客户更看重真实做工、图片可信度、发货稳定性；
- 单品过少会显得供应能力弱。

---

## 3. 不要只看 1688 销量

1688 销量只能作为弱信号。更重要的是供应链配合度。

### 3.1 供应商筛选权重

| 指标 | 权重 | 判断方式 |
|---|---:|---|
| 样品质感 | 高 | 自己买样品，看走线、边角、开孔、气味、包装 |
| 图片授权 | 高 | 是否允许用于海外 B2B 邮件、独立站、报价资料 |
| 回复速度 | 高 | 微信/旺旺是否及时，能否回答具体问题 |
| 现货稳定 | 高 | 三色是否长期有货，是否能混色 |
| 小单配合 | 高 | 10/20/50 pcs 是否能发，是否能小批量混色 |
| 买家实拍/评价 | 中 | 看差评，不只看销量 |
| 是否工厂 | 中 | 工厂更好，但初期不绝对；配合度比身份更重要 |
| 外贸经验 | 中 | 有外贸经验不一定坏，但可能价格更高 |
| 销量 | 低到中 | 太低有风险，太高也可能同质化严重 |

### 3.2 工厂 vs 贸易商

初期不必执着“必须工厂”。

更现实的排序：

1. 能稳定供货 + 愿意授权图片 + 愿意小单配合；
2. 能给真实细节图/视频 + 价格合理；
3. 之后有机会再找源头工厂或实地拜访。

---

## 4. 图片使用策略

### 4.1 第一封开发信

第一封不要重图。目标不是让对方马上下单，而是让对方回复。

建议：

- 只放产品详情页；
- 或最多附 1 张 overview image；
- 不要一封邮件塞 5-8 张图；
- 不要说 AI 处理图是未经处理的真实图。

可用口径：

```text
I can first send additional product images and real close-up photos of the sample itself.
```

### 4.2 对方回复后

再补：

- 自己拍的样品本体细节图；
- 供应商授权的适配参考图；
- 供应商授权的细节图；
- 报价和运费区间。

### 4.3 1688 图怎么用

可以用，但要先问授权。

授权话术：

```text
您好，我们准备做这款相机皮套的海外B2B客户开发，主要用于独立站和邮件给海外相机店介绍产品。

请问贵店这些产品图/细节图是你们原创实拍的吗？如果我们采购你们的产品，是否可以授权我们用于独立站产品页、海外客户邮件和报价资料展示？

我们不会冒充官方品牌，也不会用于平台铺货。可以的话麻烦您文字确认一下，谢谢。
```

如果供应商同意，保存聊天截图。

对客户称呼：

```text
Supplier reference photos for fit and detail.
```

不要称为：

```text
Our real photos
```

## 5. 市场调研看什么数据

### 5.1 公开数据

可参考：

- CIPA 相机出货数据：看数码相机和固定镜头相机整体趋势（https://www.cipa.jp/e/stats/dc.html）；
- Fujifilm / Ricoh 热门机型缺货、预售、等待名单；
- 澳洲、新加坡、马来西亚相机店是否卖 X100VI / X100V / Ricoh GR；
- Etsy、Amazon、eBay、独立站上的 half case 价格区间；
- Google Maps 上相机店、胶片店、二手相机店数量；
- Instagram 上店铺是否经常发 compact camera / film camera 内容。

图片和宣传合规可参考：

- 1688 法律声明：站内图片、照片、视频等内容存在权利归属和使用限制（https://terms.alicdn.com/legal-agreement/terms/suit_bu1_b2b/suit_bu1_b2b201802011532_36855.html）；
- 阿里巴巴知识产权保护平台：未经授权使用第三方享有著作权的照片属于著作权投诉范围（https://ipp.alibabagroup.com/policy/cn.htm）；
- 澳洲 ACCC：产品宣传不能造成虚假或误导性印象（https://www.accc.gov.au/consumers/advertising-and-promotions/false-or-misleading-claims）。

### 5.2 一人外贸最重要的数据

不要先追求大盘数据。你当前最重要的是一手市场反馈。

| 指标 | 14 天目标 | 说明 |
|---|---:|---|
| 找到潜在店铺 | 100 家 | 先悉尼，再墨尔本/新加坡 |
| 第一轮开发信 | 50 封 | 只发 A/B 类店 |
| 有效回复 | ≥1 封 | 1% 也算早期信号 |
| 愿意看图 | ≥2 家 | 哪怕不下单，也有验证价值 |
| 问 MOQ/价格/运费 | ≥1 家 | 这是强信号 |
| 愿意收样品 | ≥1 家 | 最重要的阶段性信号 |

## 6. SKU 扩展优先级

### 6.1 不建议一开始扩太散

不要同时做：

- X100VI 半套；
- X100V 半套；
- Ricoh GR 半套；
- 真皮半套；
- 背带；
- 腕带；
- 镜头盖；
- 兔笼；
- 相机包。

这样会拖慢执行，也会让网站和开发信失焦。

### 6.2 建议 SKU 路线

第一阶段：

```text
X100VI PU half case
X100VI PU half case + matching shoulder strap set
```

第二阶段，根据客户回复补：

```text
X100V half case
Ricoh GR III / GR IIIx small accessories
Leather wrist strap
Standalone leather shoulder strap
```

第三阶段再考虑：

```text
Genuine leather version
Custom model requests
Gift packaging
Small retail display set
```

### 6.3 胶片复古相机店怎么处理

可以跟进，但不要主动承诺定制。

适合加一句：

```text
If X100VI is not the right fit for your customers, I’d still be interested to hear what compact or vintage camera accessories your customers usually ask for.
```

不要写：

```text
We can customize cases for any camera model.
```

## 7. 14 天验证计划

### Day 1-2：供应商和素材

- 问 2-3 家供应商图片授权；
- 询问 10/20/50 pcs 混色价格；
- 询问是否有 X100V / Ricoh GR / 背带 / 腕带；
- 自己拍一组样品本体细节图备用；
- 确认一个可用货代区间价。

### Day 3-5：悉尼第一轮

- 从 checklist 里筛 30-50 家 A/B 类；
- 每封加一句真实观察；
- 第一封不重图，只放产品页或 1 张 overview；
- 每天记录 sent / replied / interested / not fit。

### Day 6-8：第一次跟进

- 对未回复的 A/B 类发一次简短 follow-up；
- 对回复者发真实样品细节图；
- 记录他们问什么。

### Day 9-11：扩到墨尔本或新加坡

- 如果悉尼完全无回复，换城市验证；
- 如果悉尼有回复，继续补悉尼并加新加坡小批量。

### Day 12-14：复盘

判断是否继续：

| 结果 | 判断 |
|---|---|
| 0 回复 | 文案/客户列表/品类都要调整，不急着进货 |
| 1-2 回复但无样品意愿 | 有弱信号，继续换城市和优化图片 |
| 有人要看图/问价 | 继续推进 |
| 有人愿意收样品 | 可以进入样品寄送和报价阶段 |

---

## 8. 当前建议

现在不要因为“没有完美实拍图”卡住。

当前最优动作：

1. 先问供应商图片授权；
2. 同步拍样品本体细节；
3. 用现有产品页开始发第一轮；
4. 把客户回复当作市场调研，而不是马上成交；
5. 先验证“有没有人愿意回复/看图/收样品”。

如果 50 封精准邮件没有任何反馈，不代表方向必死，但说明：

- 店铺选择不对；
- 邮件不够具体；
- X100VI 单品切入不够强；
- 或这个市场对该 SKU 的 B2B 需求很弱。

这时再决定是否扩到 X100V / Ricoh GR / 背带腕带，而不是一开始就扩散。
